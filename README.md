# growtopia
growtopia worlds database


SQLite Database of Growtopia World Names


| name | status    |
|------|-----------|
| world name | indicates the status |
| TEXT | INTEGER |

### Example
| name | status    |
|------|-----------|
| xvqw | 1 |

Means that world is not banned and its already created.


| name | status    |
|------|-----------|
| vaxt | 0 |

Means that world is banned or not yet created.
